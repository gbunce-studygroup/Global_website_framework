Icons

Usage Notes:

Two icon sets are available on all 'global website framework' sites. These are Bootstrap Glyphicons and Font Awesome. They can be added to any content box, widget, accordion or tabs compoent by following the guidelines below.

Bootstrap Glyphicons
Bootstrap Glyphicons should be wrapped in a <span></span> tag with the first class name glyphicon to declare the font family, and the second to declare the icon. For example: <span class="glyphicon glyphicon-envelope"></span> To change the size of a Glyphicon just add an inline style to the span tag. For example <span style="font-size: 30px;"
For a full list of the Bootstrap Glyphicons click here.
Font Awesome
Font Awesome icons should be wrapped in a <i></i> tag via the html input in sitecore. To declare the font family the first class name should be fa. The second is to declare the icon. For example: <i class="fa fa-phone-square"></i>
For a full list of Font Awesome icons click here, and for other information such as how to adjust the icon size click here.
